# Netflix India Content Analysis Dashboard

## Project Overview

This project analyzes Netflix India content using Microsoft Excel and presents key insights through an interactive dashboard. The analysis focuses on content growth trends, audience ratings, content categories, and the distribution of Movies and TV Shows.

## Objectives
- Analyze Netflix India content trends.
- Compare Movies and TV Shows.
- Identify yearly content addition patterns.
- Explore audience rating distribution.
- Discover the most popular content categories.

## Key Insights

### Content Growth Trend
- Netflix India experienced rapid content growth between 2017 and 2018.
- The highest number of titles was added in 2018 (333 titles).
### Movies vs TV Shows
- Movies dominate the Netflix India catalog.
- More than 90% of the available content consists of movies.
### Audience Ratings Analysis
- TV-14 is the most common audience rating.
- TV-MA and TV-PG are also widely represented.
### Category Analysis
- Drama-based content appears most frequently.
- International Movies and Independent Movies are among the most popular categories.
### Year-wise Comparison
- Movies consistently outnumber TV Shows across all years.
- TV Show additions remain relatively stable over time.
 
## Tools & Techniques Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization
- KPI Reporting
  
## Project Outcome

Developed a KPI-driven Excel dashboard that transforms raw Netflix content data into actionable insights, helping stakeholders understand content trends, audience preferences, and category performance through interactive visualizations.
